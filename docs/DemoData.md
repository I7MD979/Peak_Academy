# بيانات تجريبية — Peak Academy

## تشغيل السيد (مرة واحدة أو بعد إعادة ضبط قاعدة البيانات)

### 1) إعداد المفاتيح (مرة واحدة)

```bash
cd backend
npm run setup:env
```

الصق **service_role** من [Supabase → Settings → API](https://supabase.com/dashboard/project/hpczrdvaeazrrrzgtatl/settings/api)  
(ليس anon ولا JWT Secret)

### 2) تشغيل السيد

```bash
npm run seed:demo
```

هذا الأمر ينشئ حسابات الدخول ثم يملأ المنصة ببيانات تجريبية.

## حسابات الدخول

| الدور | البريد | كلمة المرور |
|--------|--------|-------------|
| مشرف | admin@peak.com | Admin123! |
| معلم | teacher@peak.com | Teacher123! |
| طالب | student@peak.com | Student123! |
| ولي أمر | parent@peak.com | Parent123! |

## ما الذي يُضاف؟

### المعلم (`teacher@peak.com`)
- مواد: رياضيات، فيزياء، كيمياء
- نبذة مهنية وصفوف (تاني/تالتة ثانوي)
- جلستان (قادمة + مكتملة)
- طلب سحب معلّق (450 ج.م)
- غرفتا مذاكرة مرتبطتان بمادته

### الطالب (`student@peak.com`)
- صف: تالتة ثانوي
- تجربة مجانية 30 يوماً لغرف المذاكرة
- مسجّل في الجلستين التجريبيتين
- عضو في غرفة رياضيات نشطة + رسائل دردشة
- سؤالان (واحد بلا إجابة، واحد مُجاب)

### ولي الأمر (`parent@peak.com`)
- مربوط تلقائياً بالطالب التجريبي
- يمكنه فتح لوحة ولي الأمر والتقارير مباشرة

### الإدارة (`admin@peak.com`)
- كوبون تجريبي: `DEMO20` (خصم 20%)

## أين تجرب؟

| الميزة | المسار |
|--------|--------|
| غرف المذاكرة (معلم) | `/teacher/study-rooms` |
| غرف المذاكرة (طالب) | `/student/study-rooms` |
| جلسات المعلم | `/teacher/sessions` |
| لوحة ولي الأمر | `/parent/dashboard` |
| السحوبات (إدارة) | `/admin/withdrawals` |
| العروض (إدارة) | `/admin/promotions` |

## إعادة التشغيل

السكربت آمن للتكرار (يستخدم معرّفات ثابتة تبدأ بـ `demo-`). شغّل:

```bash
npm run seed:demo
```
